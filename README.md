# Do Automatic Image Similarity Metrics Reflect Human Preferences? A Human Study of ChatGPT and Gemini Image Reconstruction

[![DOI](https://zenodo.org/badge/1061989386.svg)](https://doi.org/10.5281/zenodo.21223838)

This repository contains the code and data for the paper **"Do Automatic Image Similarity Metrics Reflect Human Preferences? A Human Study of ChatGPT and Gemini Image Reconstruction"**.

## Overview

Text-to-image systems are frequently benchmarked using automatic similarity scores, but real-world deployment relies heavily on human perception. This controlled pilot study investigates how three widely used perceptual metrics align with human preferences when evaluating image variants generated by ChatGPT (DALL-E) and Gemini.

The repository provides the complete end-to-end workflow, including:
*   **Anonymized human rankings:** Data from 10 raters evaluating 15 sets of reference images.
*   **Image variants and captions:** The generated images (gg, cg, gc, cc variants) and their corresponding text prompts.
*   **Evaluation code:** Scripts to compute automatic metrics and statistical correlations (Kendall's W, Spearman, Pearson).

## Evaluated Metrics

The codebase computes and analyzes the following metrics against ground-truth human orderings:
*   **LPIPS** (Learned Perceptual Image Patch Similarity) via the AlexNet backbone.
*   **MS-SSIM** (Multi-Scale Structural Similarity).
*   **CLIP Cosine Similarity** (OpenAI ViT-B/32 encoder).

## Key Findings

*   **Intra-Model Consistency:** Intra-model consistency is stronger than inter-model consistency. Raters heavily favored Gemini-generated images prompted by Gemini captions (the "gg" variant), ranking them first 44.0% of the time.
*   **Human Agreement:** Human raters demonstrated moderate agreement ($\overline{W}=0.40$) on the evaluated image sets. 
*   **Metric Alignment:** CLIP showed the strongest (yet modest) alignment with human rankings with a mean Spearman correlation of 0.36. LPIPS and MS-SSIM correlated weakly, confirming that no single automatic metric acts as an adequate proxy for human judgment in this context.
